## How every command works: Input → Process → Output

Every command follows a strict execution pipeline. No vague questions. No generic advice.

**Input:** The command tells you exactly what data to paste and from which platform. "Paste Meta Ads Manager > Campaigns tab: spend, purchases, ROAS, CPA, CPM, CTR, CPC (last 30 days)." It never asks "how is your marketing going?" It asks for the specific dashboard data it needs to run calculations.

**Process:** The command takes your raw data and does the math. It calculates your break even ROAS, compares your CPA against your vertical benchmark, diagnoses the root cause of underperformance using proven frameworks, and sizes every opportunity in dollars per month. You see the calculations. You confirm the numbers look right.

**Output:** The command produces a finished deliverable you deploy immediately. Not a recommendation. Not a list of ideas. The actual campaign structure, flow architecture, pricing model, or action plan with deployment instructions telling you exactly where to go in the platform and what to configure.

Every output includes: the deployable deliverable, where to deploy it, which profit lever it targets, benchmark context, and the estimated dollar impact of deploying it.
