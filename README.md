### How Each Command Works

1. Open Claude Code
2. Select a command from the /ss: menu
3. Paste and replace [VARIABLES] with your store data
4. Execute and get a deployable output
5. Deploy directly to Meta, Google, TikTok, Klaviyo or Shopify

Every command logs activity to your private dashboard at localhost:3000.
