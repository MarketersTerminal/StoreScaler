# Store Scaler: Execution Map

24 categories. 143 commands. 8 groups. 5 profit levers covered.

Every command produces a finished, deployable output. Each is either a one-time **Setup** or a recurring **Ongoing** action (weekly, monthly, quarterly, or per-event). The `/ss:` prefix is the command namespace inside Claude Code.

Roughly 40% of commands are Setup (one-time foundation). 60% are Ongoing (recurring optimization).
